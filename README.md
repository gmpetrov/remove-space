# remove-space

> Replace filenames's whitespaces by dashes

```
ls dir // output 'filename with whitespaces'
remove-space dir
ls dir // output 'filename-with-whitespaces'
```

# Install

```
yarn global add @gmpetrov/remove-space
// or npm install -g @gmpetrov/remove-space
```

# Usage

```
remove-space [dir_path]
```
